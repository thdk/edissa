# Edissa

## Description

Reservation management platform built with [Nest](https://github.com/nestjs/nest).

Built as hobby project to gain hands-on experience with:
  - nest
  - typeorm
  - google cloud spanner 
  - postgresql
  - grpc
  - terraform
  - microservices
  - dev containers
